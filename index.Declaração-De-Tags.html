As tags HTML h1 a h6 são usadas para criar cabeçalhos ou títulos em uma página da web. Elas representam diferentes níveis de importância e hierarquia, onde h1 é o nível mais alto (o título principal da página) e h6 é o nível mais baixo.


Temos tambem a tag chamada de P , uma tag usada para é usada para definir um parágrafo em um documento HTML. Um parágrafo é uma unidade básica de texto que consiste em uma ou mais sentenças relacionadas que expressam uma ideia ou tópico específico.
